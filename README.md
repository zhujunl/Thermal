## 中正人脸闸机

这个工程中，通过策略模式去适配的各种不同的设备，或为相同设备表现出不同的结果，通过ValueUtil中的sign字段进行控制，于ConfigManager中的isXXX方法中进行功能开关控制。

界面及说明请看使用说明书。

### 版本说明

    XH：旭辉设备，带二型测温摄像头，已废弃
    XH_N：旭辉设备，带五型测温摄像头，已废弃
    XH_C：旭辉设备，不带测温摄像头，通用版本
    MR870：已废弃
    MR870A：双面屏设备，横屏适应，带身份证，带指纹，已废弃
    ZH：中控设备（似乎），仅此一台，已废弃
    TPS980P：天波设备，带测温摄像头，注意此台设备与其他天波不带摄像头版本的设备的摄像头配置参数不同，需要特别区分
    TPS980P_C：天波设备，通用版本，不使用其他外设
    TPS980P_F：天波设备，带中正身份证阅读器串口
    TPS980P_M：天波设备，带天波IC卡阅读器，此版本因设备从未到位过，所以未测试IC功能能否正常使用
    TPS980P_T：天波设备，带测温摄像头，但是未使用到测温摄像头，由于摄像头参数问题，单独开了一个
    TPS980P_FA：天波设备，带天波身份证阅读器
    MR890：门口那台闸机上的新款设备，仅此一台，已废弃
    MR860DZ：带微波雷达
    PAD：李顺勇拿来的一台广告屏设备，然后就没然后了，横屏适应

### 版本对应项目

    富生电器：大门口对应XH_C，侧门车道对应TPS980P_F
    健康屋：TPS980P_F和TPS980P_FA
    老年活动中心：TPS980P_F
